# Hearloop: Quick Reference Context

**Use this for 30-second refreshes. For full context, see `copilot-instructions.md`.**

## What Is This?
Voice feedback widget platform. Users record 5-sec clips → transcribed & analyzed → webhook sent to partner.

## Where Is What?
```
api/          Backend: Fastify + Node + TS
web/          Frontend: Next.js + React
packages/db   DB migrations
packages/types Shared types
```

## Tech Stack (One Line Each)
- DB: PostgreSQL (Supabase)
- Queue: Redis + BullMQ (Upstash)
- Storage: Cloudflare R2
- Speech: Groq Whisper
- Analysis: Claude Haiku (JSON)

## Key Files & What They Do
| Path | Does What |
|------|-----------|
| `api/src/index.ts` | Fastify setup |
| `api/src/lib/groq.ts` | Whisper wrapper |
| `api/src/lib/claude.ts` | Analysis wrapper |
| `api/src/lib/queue.ts` | Job queue |
| `api/src/jobs/*.ts` | Async workers |
| `api/src/routes/*.ts` | API endpoints |
| `web/app/capture/[token]/page.tsx` | Recording page |
| `web/components/widget.js` | Embedded widget |

## Session Lifecycle
1. Widget created (token issued)
2. User records → stored in R2
3. Job: transcribe (Groq) → analyze (Claude)
4. Job: deliver webhook (HMAC signed)

## Auth 3 Ways
- **API Keys:** Partner server calls (`X-API-Key`)
- **Session Tokens:** Widget requests (scoped to one session)
- **Webhooks:** HMAC validation (partner verifies signature)

## To Add Something New
**New API endpoint:** `api/src/routes/` → export `setupMyRoute(app)` → call in `index.ts`  
**New job:** `api/src/jobs/my-job.ts` → export handler → register in `queue.ts`  
**New schema:** `packages/db/migrations/` → new SQL file → bump number

## Run Commands
```bash
npm run dev       # Start all (turbo)
npm run build     # Build all
```

## Important Concepts (Know These)
- **Monorepo:** Turbo orchestrates builds
- **Multi-tenant:** Each partner has secret API key
- **Queue:** BullMQ handles async tasks (transcription, analysis)
- **Topic Taxonomy:** Fixed enum (friendliness, wait time, service quality, etc.)
- **Session State:** created → processing → completed

## ENV Vars Needed
Supabase (DB_URL), Upstash Redis (REDIS_URL), Groq (GROQ_API_KEY), Claude (ANTHROPIC_API_KEY), Cloudflare R2 (bucket + credentials)
